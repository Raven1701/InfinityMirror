# InfinityMirrorSMoptical illusion with infinity leds between the mirrors. Non-commercial open-source project.
<p style="text-align: center;">
It is a simple and fast project for classes "Microsystems". It's app for remote control my strip LED in InfinityMirror.<p>&nbsp;</p>
I use this <p style="text-align: center;"><a href="https://github.com/QuadFlask/colorpicker">ColorPicker</a></p>.</p>
